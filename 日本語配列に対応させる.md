BusyBoxの `loadkmap` コマンドを使用することで対応する。

まず、`ckbcomp` を使用してJIS配列のテキスト形式キーマップを標準出力に書き出す。パイプでテキスト形式のキーマップを受け取り、`loadkeys` を使用して、BusyBoxのバイナリ形式に変換する。
```
ckbcomp -model jp106 -layout jp | loadkeys --bkeymap > /path/to/rootfs/etc/jp106.bmap
```

> ckbcomp(1)
> https://manpages.debian.org/unstable/console-setup/ckbcomp.1.en.html

その後、`/init` などの [[initramfs]] 起動時に動作するシェルスクリプトに、下記の内容を記述する。
```
if [ -f /etc/jp106.bmap ]; then
    loadkmap < /etc/jp106.bmap
fi
```
