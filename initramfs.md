cpio アーカイブなどの最小構成でメモリ上に展開することで、用意した最低限の環境を用意した際のメモリ上のファイル構造の事。
initramfs では、`/` の中身を消去した後に、実際のドライブを `/` にマウントして、一般的なユーザーランドの [[rootfs]] を作成する。

カーネル起動直後では、ハードディスクがマウントされていないためメモリ上のみでハードディスクへのマウントを行う必要があるため、このようなFile Systemが存在する。（initramfs の fs は File System の略）

実際の initramfs でやることはハードディスクへのマウントがメイン。
